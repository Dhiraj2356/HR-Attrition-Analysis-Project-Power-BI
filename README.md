## Project Overview
 This project analyzes employee attrition data to identify key factors contributing to employee turnover. The dashboard helps HR teams understand attrition trends and make data-driven retention decisions.

## Objectives

Measure overall attrition rate
Identify departments and roles with high attrition
Analyze attrition by age, gender, salary, and experience
Support HR decision-making with actionable insights

##  Tools & Technologies

Power BI Desktop
DAX
Excel (for validation)

## Key KPIs Created

Total Employees
Attrited Employees
Attrition Rate (%)
Average Age
Average Monthly Income

## Data Cleaning & Preparation

Removed duplicate employee records
Handled missing values
Created age groups and salary slabs for analysis

## Key Insights

Higher attrition observed in specific departments and job roles
Employees with lower salary bands showed higher attrition
Early-career employees had higher attrition rate
